Search Engine Optimisation

For a better search optimisation of a website we must use seo rules or seo format or seo techniques for a better website optimisation

1. write a good title to catch the search requirements of an users

a good title gives a good understanding of the website we want to go

2. write a meta description

meta description includes the actual 2 lines of the whole content for the google to know what does our website actually contains

3. write a good url

A good url gives a better acknowledgement of the website

4. write meta keyword

a meta keyword block conists of keywords which are present in the website for seo understanding

5. write a meta author

an author determines the person who actually wrote the website or the content of the website we are using

6. give a favicon

a favicon is an icon given to the website for recognisation of the website
